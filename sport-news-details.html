Sport News

Developed a dynamic Sports News platform in Drupal 10, displaying the latest articles per category with custom views, pop-up notifications for inactive users, and a fully responsive, visually appealing design.

Developed a dynamic Sports News platform in Drupal 10 to allow administrators to efficiently create, categorize, and display the latest sports articles. The system features a custom view and a pop-up block to engage users with the newest sports updates, all presented in a responsive and visually appealing design.

  • Deliver the latest sports news per category in a way that is clear and attractive.
  • Ensure user engagement by prompting inactive visitors with relevant news highlights.
  • Make the solution responsive and visually appealing, fitting both desktop and mobile devices.
  • Implement custom styling to enhance readability, emphasize categories, and highlight key images.

  • Created a custom content type “Sports News” with fields: Title, Image, Description, and Category.
  • Built a taxonomy “Categories” to organize news by sports types (e.g., Football, Basketball, Tennis).
  • Developed a custom view displaying the latest article per category, including title, image, description, and category badge, available as a page and a block.
  • Implemented a custom block module (custom_news_block) that shows the view content in a pop-up modal after 30 seconds of user inactivity using JavaScript.

Key Features

  • Custom Block & Pop-up
  • Smooth Animations
  • Taxonomy-Based Categories
  • Responsive Design
  • Stylized Content
